rucio-conveyor-finisher

usage: rucio-conveyor-finisher [-h] [--run-once]
                               [--total-threads TOTAL_THREADS]
                               [--db-bulk DB_BULK] [--bulk BULK]
                               [--sleep-time SLEEP_TIME]
                               [--activities ACTIVITIES [ACTIVITIES ...]]

Conveyor is a group of daemons to manage file transfers. The conveyor-finisher
is the responsible to update Rucio internal state after the transfer has
finished.

optional arguments:
  -h, --help            show this help message and exit
  --run-once            One iteration only
  --total-threads TOTAL_THREADS
                        Concurrency control: total number of threads per
                        process
  --db-bulk DB_BULK     Bulk control: number of transfers per db query
  --bulk BULK           Bulk control: number of requests
  --sleep-time SLEEP_TIME
                        Seconds to sleep if few requests
  --activities ACTIVITIES [ACTIVITIES ...]
                        Explicit list of activities to include
