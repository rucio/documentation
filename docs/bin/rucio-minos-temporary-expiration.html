Running rucio-minos-temporary-expiration

usage: rucio-minos-temporary-expiration [-h] [--run-once] [--threads THREADS]
                                        [--bulk BULK]
                                        [--sleep-time SLEEP_TIME]

This special type of minos daemon lists the expired TEMPORARY_UNAVAILABLE
replicas and puts them back into AVAILABLE state.

optional arguments:
  -h, --help            show this help message and exit
  --run-once            Runs one loop iteration
  --threads THREADS     Concurrency control: number of threads
  --bulk BULK           Bulk control: number of requests per cycle
  --sleep-time SLEEP_TIME
                        Seconds to sleep if few requests
